### SunamoPInvoke

P/Invoke wrapper library for Windows API (WinAPI) interop in .NET.

Provides managed wrappers for common Windows API functions including global keyboard hooks, file operations (recycle bin), clipboard management, process token manipulation, and shell icon extraction.
