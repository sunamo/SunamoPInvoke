# NuGet Alternatives to SunamoPInvoke

This document lists popular NuGet packages that provide similar functionality to SunamoPInvoke.

## Overview

P/Invoke utilities

## Alternative Packages

### PInvoke.Kernel32
- **NuGet**: PInvoke.Kernel32
- **Purpose**: Windows API P/Invoke signatures
- **Key Features**: Type-safe Win32 imports

### PInvoke.User32
- **NuGet**: PInvoke.User32
- **Purpose**: User32.dll imports
- **Key Features**: Window management, input

### Vanara
- **NuGet**: Vanara.PInvoke
- **Purpose**: Comprehensive Windows API
- **Key Features**: All Windows APIs, COM support

### pinvoke.net
- **NuGet**: null
- **Purpose**: P/Invoke reference site
- **Key Features**: Community signatures database
